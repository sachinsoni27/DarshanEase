# 🕉️ DARSHAN EASE - Temple Discovery Platform

> A comprehensive platform for pilgrims to search temples and plan their spiritual journeys



## 📖 About

DARSHAN EASE is a web application designed to help devotees discover temples and plan their pilgrimage routes. Whether you're looking for ancient temples, specific deities, or planning a spiritual journey, DARSHAN EASE makes it easy to find and explore sacred destinations.

## ✨ Features

- 🔍 **Temple Search**: Search for temples by name, location, or deity
- 📍 **Location-Based Discovery**: Find temples near you
- 🗺️ **Pilgrimage Planning**: Plan multi-temple visits and routes
- 📱 **Responsive Design**: Works seamlessly on mobile and desktop
- 🕉️ **Detailed Information**: Get temple history, timings, and visiting guidelines

## 🛠️ Tech Stack

- **Frontend**: TypeScript, HTML, CSS
- **Deployment**: Vercel (Continuous Deployment)
- **Development**: Built using Antigravity (Google's IDE)
- **Version Control**: Git & GitHub
